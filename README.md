# calculator v1.0

This program can parse a string entered by the user. It calculates the mathematical result of the string.

Working features:
- Numbers
- Operators (+,-,*,/,^)
- Operator priority
- Brackets
- Functions

- Only use double type

- Show a list of the implemented functions
- Add modulo %
